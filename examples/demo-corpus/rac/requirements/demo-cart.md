# Requirement: Cart

## Status

Accepted

## Problem

Shoppers need to add items to a cart and review it before checkout. The cart
must reflect quantity and price changes so a shopper trusts the total.

## Requirements

- A shopper can add an item to the cart and see it listed.
- A shopper can change an item's quantity and see the total update.
- A shopper can remove an item from the cart.

## Acceptance Criteria

- Adding an item shows it in the cart with its price.
- Changing a quantity updates the line total and the cart total.
